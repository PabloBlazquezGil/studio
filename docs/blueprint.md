# **App Name**: PabloBlázquezGil

## Core Features:

- Animated Typography Logo: A dynamically animated typographic logo in the navbar (name: Pablo Blázquez Gil).
- Full-screen Video Hero Section: A full-screen background video with dark overlay and animated text.
- Interactive Photo Gallery: Responsive image grid with scaling and overlay on hover, displaying title, category, and short description.
- Parallax Video Cards: Video cards with a parallax mouse effect and floating play button that follows the cursor.
- Asymmetric About Section: Asymmetric layout with a parallax text background and revealing author image.
- Infinite Scroll Carousel: Infinite carousel of client logos in grayscale that regain color on hover.
- Project Detail Overlay: Overlay for immersive project details including metadata and a mixed media gallery with lightbox functionality.

## Style Guidelines:

- Primary color: Fern Green (#5E9C76) for an elegant accent.
- Background color: Thunder Dark (#0A0F0D), a deep greenish black, sets a dark cinematic tone.
- Accent color: Thunder Light (#E6EBE8), a fog white, provides contrast and highlights.
- Headline font: 'Playfair Display' (serif) for titles, adding a touch of editorial and classic style.
- Body font: 'Manrope' (sans-serif) for readability and modern feel.
- Use Lucide React icons with thin, elegant lines for a minimalist aesthetic.
- Incorporate a film grain overlay across the site to add texture.
- Design a custom cursor that mimics a camera shutter, contracting on click.
- Implement smooth scroll behavior to enhance user experience.
- Use fade-in animations on scroll for each section to create a smooth and engaging experience.
